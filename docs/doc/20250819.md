 什么是数据治理，数据质量
数据治理是组织中涉及数据使用的一整套管理行为。
由企业数据治理部门发起并推行，关于如何制定和实施针对整个企业内部数据的商业应用和技术管理的一系列政策和流程。    
数据的质量直接影响着数据的价值，并且直接影响着数据分析的结果以及我们以此做出的决策的质量。
数据治理即提高数据的质量，发挥数据资产价值。
